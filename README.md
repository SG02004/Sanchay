# Sanchay

Project structure:

```text
sanchay/
  backend/   Express + MongoDB API
  frontend/  React + Vite app
```

Run the apps from the repo root:

```bash
npm run frontend:dev
npm run backend:dev
```

Or run them directly inside each folder:

```bash
cd frontend
npm install
npm run dev
```

```bash
cd backend
npm install
npm run dev
```
